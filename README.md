# Ghost Shooter

A fast-paced 2D top-down survival shooter game built with **Cocos Creator** and **TypeScript**. 

## 🎮 Features
- **Fluid Controls:** Smooth 2D top-down movement, mouse-based aiming, shooting, and a vector-bounded dash mechanic.
- **Dynamic Enemy AI:** Enemies feature real-time player tracking and boundary collision/bounce logic using vector math.
- **State Management:** Robust game state controller handling Main Menu, Game Over, and seamless scene restarting.
- **UI & Progression:** Includes a real-time Minimap/Radar, visual feedback (Tween animations), and persistent High Score tracking using local storage.
- **Optimized Physics:** Advanced Box2D collision handling to prevent multi-hit frame bugs and safe deferred node destruction.

## 🛠️ Technologies Used
- **Game Engine:** Cocos Creator (v3.x)
- **Language:** TypeScript
- **Physics Engine:** Built-in Cocos 2D Physics (Box2D)

## 🕹️ Controls (Expected)
- **W, A, S, D:** Move
- **Mouse:** Aim
- **Left Click:** Shoot
- **Right Click** Skill Dash
